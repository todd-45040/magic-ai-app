# Visual Brainstorm Anatomical Artifact Suppression Patch

## Scope
Targeted UI/service prompt hardening only. No auth, billing, telemetry, schema, routing, or AI infrastructure changes.

## Changes
- Added a global anatomical artifact suppression block to Visual Brainstorm image prompts.
- Reinforced complete-performer staging, clean centered composition, and single-subject prop showcase defaults.
- Strengthened negative guidance against floating arms, detached hands, phantom assistants, cropped body fragments, malformed anatomy, horror-style appendages, and disembodied audience interaction.
- Tuned refinement presets that previously encouraged floating audience hands or horror-style artifacts.

## Expected Result
Visual Brainstorm outputs should bias toward professional stage photography with complete human figures, clean performer staging, and fewer hallucinated appendages.
