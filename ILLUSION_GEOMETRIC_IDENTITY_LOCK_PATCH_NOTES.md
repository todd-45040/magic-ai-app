# Illusion Blueprint Geometric Identity Lock Patch

Implemented a geometry-first continuity layer for Illusion Blueprint generation.

## Scope
- Targeted Illusion Blueprint prompt orchestration only.
- No auth, billing, telemetry, schema, global state, or routing changes.

## Added
- Geometry profile extraction for seeded apparatus forms.
- Silhouette lock rules for dog-house, cabinet/box, rope/ring, and generic apparatus families.
- Proportion lock rules shared by Blueprint A/Concept A and Blueprint B/Concept B.
- Component inheritance rules so operational states cannot redesign the apparatus.
- Geometric identity requirements injected into blueprint and render prompts.

## Intended result
- Stronger roofline continuity.
- Stronger base/platform continuity.
- Stronger facade and opening continuity.
- Fewer blueprint/render reinterpretations.
- Better persistence of the selected apparatus as the same physical object across paired outputs.
