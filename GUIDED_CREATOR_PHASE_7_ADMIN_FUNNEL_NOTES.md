# Guided Creator Phase 7 — Admin Funnel Visibility

## Summary
Adds admin dashboard visibility for Guided Creator onboarding performance.

## Updated files
- `api/adminKpis.ts`
- `components/AdminOverviewDashboard.tsx`

## Metrics added
- Views
- Path selected
- Generation completed
- First idea saved
- Skip rate
- Median time to first save
- Conversion from Guided Creator session to checkout started

## Notes
- Metrics are calculated from `analytics_events` generated by the Guided Creator flow.
- Checkout conversion counts unique users who viewed Guided Creator and later triggered `upgrade_checkout_started` or `checkout_started` within the selected dashboard window.
- This patch does not modify billing, auth, Stripe webhooks, or LiveRehearsal.

## Validation
- `npm run build` passed.
- Targeted TypeScript parse/check was attempted; remaining reported issues were pre-existing project configuration/type issues around `_auth` union narrowing and `import.meta.env`, not introduced by this patch.
